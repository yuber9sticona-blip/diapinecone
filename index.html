Concepto

¿Qué es un Vector?

Un vector es una lista de números que encapsula el significado de cualquier dato — texto, imagen, audio o video. Modelos como OpenAI Embeddings o Sentence-Transformers los generan automáticamente.

  • "Gato" → [0.21, 0.87, 0.14, 0.65…]
  • "Felino" → [0.22, 0.85, 0.16, 0.63…] ← ¡Muy cerca!
  • "Avión" → [0.91, 0.03, 0.72, 0.11…] ← Lejos.
🗺️ Analogía del Mapa

Las ideas similares viven cerca en un espacio matemático multidimensional. Pinecone es el GPS que navega ese espacio en milisegundos.

Funcionamiento

¿Cómo Funciona?

Pinecone opera en 4 etapas claras. Desde que tienes un documento hasta obtener una respuesta inteligente.

1
Generar Embeddings
Tu texto/imagen pasa por un modelo (OpenAI, Cohere, HuggingFace) y se convierte en un vector de 768 o 1536 dimensiones.
2
Crear Índice (Index)
Se define el contenedor en Pinecone: dimensiones del vector, métrica de distancia (coseno, euclidiana) y namespace.
3
Upsert — Subir Vectores
Se suben los vectores junto con metadatos (id, fuente, fecha, categoría) para filtrado posterior.
4
Query — Consultar
La pregunta del usuario se vectoriza y Pinecone devuelve los K vectores más similares en ~50ms usando ANN (Approximate Nearest Neighbor).
05 / 09
Implementación

Código Real en Python

Así se usa Pinecone en producción con OpenAI Embeddings. Son apenas ~20 líneas para tener búsqueda semántica funcional.

  • pinecone-client: SDK oficial para Python, Node.js y más.
  • Upsert: Sube vectores con ID único y metadatos opcionales.
  • Query: Devuelve los K vectores más cercanos con score de similitud.
  • Filtros: Filtra por metadatos antes de buscar (ej: solo documentos legales del 2024).
⚡ Resultado

Con este código, cualquier pregunta del usuario se convierte en búsqueda semántica sobre miles de documentos en <50ms.

# Instalar: pip install pinecone openai
from pinecone import Pinecone, ServerlessSpec
from openai import OpenAI

# 1. Inicializar clientes
pc = Pinecone(api_key="TU_API_KEY")
ai = OpenAI(api_key="TU_OPENAI_KEY")

# 2. Crear índice serverless
pc.create_index(
    name="documentos",
    dimension=1536,
    metric="cosine",
    spec=ServerlessSpec(cloud="aws", region="us-east-1")
)
index = pc.Index("documentos")

# 3. Generar embedding y subir
texto = "El contrato fue firmado en enero 2024"
vector = ai.embeddings.create(
    model="text-embedding-3-small",
    input=texto
).data[0].embedding

index.upsert(vectors=[{
    "id": "doc_001",
    "values": vector,
    "metadata": {"tipo": "legal", "año": 2024}
}])

# 4. Buscar por similitud semántica
pregunta = "¿Cuándo se firmó el acuerdo?"
q_vec = ai.embeddings.create(
    model="text-embedding-3-small",
    input=pregunta
).data[0].embedding

resultados = index.query(
    vector=q_vec,
    top_k=3,
    filter={"tipo": {"$eq": "legal"}},
    include_metadata=True
)

# → Score: 0.97 | doc_001 | "El contrato fue firmado…"
